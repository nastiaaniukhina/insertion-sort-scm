# Insertion Sort — C++

Консольний застосунок, що демонструє сортування вставками масиву символів.

## Опис
Програма генерує випадковий масив із 10 символів і сортує 
його методом вставок, виводячи кожен крок сортування.

## Збірка
g++ main.cpp -o sort

## Запуск
./sort        (Linux/Mac)
sort.exe      (Windows)
